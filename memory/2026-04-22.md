## 🔧 Chrome DevTools 附加模式 - 重大进展

### MCP Server 配置（已写入 openclaw.json）
```json
"mcp": {
  "servers": {
    "chrome": {
      "command": "npx",
      "args": ["chrome-devtools-mcp@latest", "--browserUrl", "http://127.0.0.1:9222"]
    }
  }
}
```
- Chrome 调试端口：9222
- 需要在 chrome://inspect/#remote-debugging 勾选 "Allow remote debugging"
- Gateway 已重启生效

### browser profile 配置（openclaw.json）
```json
"browser": {
  "enabled": true,
  "defaultProfile": "user",
  "profiles": {
    "user": {
      "driver": "cdp",
      "cdpUrl": "http://127.0.0.1:9222",
      "color": "#FF4500"
    }
  }
}
```

### ⚠️ 关键限制（重要！）
1. **无法新建标签页** - `chrome__new_page` 会报 "HTTP Not Found"，这是 OpenClaw 安全策略限制
2. **无法主动导航** - `browser navigate` 会被网关拦截
3. **可以执行 JS 跳转** - `window.location.href` 在已有标签页内跳转可以绕过导航限制
4. **可以操作现有页面** - 快照、截图、JS执行、点击、填表都没问题
5. **MCP chrome 连接方式不同** - 使用 websocket 直接连 Chrome，而非通过 CDP URL
